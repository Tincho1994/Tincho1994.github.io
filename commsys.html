Custom Sphero SPRK Swarm Communication System

Summer 2017
Cornell Autonomous Systems Lab

Spheros use bluetooth to receive commands. A single dongle could reliably connect to and communicate with three Spheros with minimal latency. To scale control for a swarm, a system for sending commands to multiple groups was implemented.

Due to the bluetooth hardware constraints multiple computers would be needed to run a swarm. Easy and quick setup was necessary to feasibly run experiments so system flexibility and robustness were major design focuses.

The computers used to command a sub-swarm launch an instance of a python program to connect to each Sphero. These programs send commands that are piped by another python program that is listening for data broadcast over wifi by the computer running the control program. The system is designed to run either as standalone or integrated with ROS and could be run on a Linux machine or Raspberry Pi.
