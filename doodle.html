Doodle: A Robotic Drawing Companion

Summer - Fall 2017
Personal Project/ Master's of Engineering Project
Ithaca, NY

This is a current ongoing capstone project for my graduate program in mechanical engineering The goal of this project is to create a robot that would be able to observe a childâ€™s drawing and draw with them. The vision is for this robot to enhance drawing as a recreational and creative outlet for children through human-robot cooperation. A main focus of this project is to create a social robot that is able to communicate using nonverbal gestures with it's user.

With the current system design a user would draw on a piece of paper and then slide the paper under a frame with an overhead camera. The camera uses computer vision to take in the drawing and generate commands for a smaller mobile robot that will then run around the paper and draw before signaling to the user that it is now their turn.

The basis for the vision system was built using openCV. The current proof of concept model detects the canvas space of a piece of paper and contours on the drawing. Current work includes using that data to generate paths for a drawing robot as well as using the overhead camera to localize a small mobile robot that will draw on the paper. We are also developing our own system for the mobile robot, sourcing parts and designing the chassis and relevant mechanisms such as a pen up - pen down actuator. The robot is currently designed to run using a Raspberry Pi Zero W and have it's own onboard power supply. The mobile robot will use bluetooth to communicate with the Raspberry Pi 3 that is running the vision and control system.
