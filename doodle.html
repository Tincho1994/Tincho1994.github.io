Doodle: A Robotic Drawing Companion

Summer - Fall 2017
Master's of Engineering Project

Previous Next

The goal of this project is to create a robot that could serve as a drawing companion for children. The vision is for this robot to enhance drawing as a recreational and creative outlet for children through human-robot cooperation.

The system is composed of an overhead localization system and a mobile robot platform. The user uses a pen with a marker on top to signal to the robot where to follow. The mobile robot receives motion commands from the vision system as it follows the user. The pen has two tags on either side which signal to the robot whether to retract or deploy a pen which is used to draw as the robot moves.

A Raspberry Pi 3 is used to control the overhead localization with the use of a camera and openCV to track markers. The mobile robot platform uses a Pi Zero W to receive commands from the Pi 3 via bluetooth. The mobile robot platform comes with it's own power source and uses two continous rotation servos to drive it's wheels and a servo for pen actuation.
